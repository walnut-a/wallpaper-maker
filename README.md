# 手机壁纸生成器

一个偏克制、偏成品感的参数化手机壁纸工具。  
打开页面就能直接看到一张壁纸，再通过预设、配色、种子和高级参数继续细调。

## 在线使用

访问 GitHub Pages：

[https://walnut-a.github.io/wallpaper-maker/](https://walnut-a.github.io/wallpaper-maker/)

## 特性

- 单文件应用：全部逻辑都在 `index.html`，没有构建步骤，没有后端依赖
- 打开即用：首屏直接生成成品壁纸，不需要先配置一堆参数
- 四种风格预设：`极简秩序感`、`大色块构成`、`科技纹理感`、`有机生成感`
- 双层参数结构：默认只放高频控制，细调项收进高级参数里
- 尺寸可控：支持常见手机尺寸预设，也支持自定义宽高
- 种子复现：相同参数和相同种子会生成同一张图
- 实时预览：修改参数后立刻重绘，不需要额外点击“应用”
- 本地导出：一键导出高清 `PNG`
- 静态部署友好：直接放到 GitHub Pages 就能用

## 风格预设

- 极简秩序感：大形体退后，底纹很轻，更接近被光擦过的安静构图
- 大色块构成：几团有重量的软形体互相压住，重点是体积感，不是硬块拼贴
- 科技纹理感：深色场、模糊光团、轻蜂窝底纹，整体更像带秩序的光学材质
- 有机生成感：被拖开的柔软色团和轻微流向变化，氛围更自然

## 可调内容

默认区主要负责高频操作：

- 风格预设切换
- 尺寸预设和自定义宽高
- 调色板
- 密度
- 复杂度
- 随机种子
- 重新生成
- 随机参数
- 重置当前预设
- 导出 `PNG`

高级区负责更细的画面控制，不同预设会显示不同参数组合，主要包括：

- 主形体数量
- 边缘软化
- 光晕半径
- 底纹强度
- 冷暖偏移
- 颗粒细度
- 块面偏移
- 明暗反差
- 雾化程度
- 蜂窝尺度
- 暗角强度
- 漂移幅度
- 流向拉伸
- 纱幕透明度

## 设计方向

这个项目不是在做“参数很多的几何练习器”，而是在做“能产生成品壁纸的静态工具”。

目前的视觉重点是：

- 少量主形体，而不是满屏小元素
- 更像光和体积的颜色关系，而不是平铺填色
- 更软的边缘和更轻的纹理
- 规则底纹退后，主体氛围往前
- 桌面端左侧控制、右侧预览；移动端上预览、下参数

## 技术实现

- 纯 `HTML + CSS + JavaScript`
- 使用 `Canvas` 统一负责预览和导出
- 预览与导出共用同一套绘制逻辑，避免“看到的”和“导出的”不一致
- 支持 `file://` 和普通静态站点环境

## 本地使用

### 直接打开

直接双击 `index.html` 即可使用。

### 本地静态服务

如果你想用本地服务预览，可以运行：

```bash
python3 -m http.server 8123
```

然后访问：

```text
http://127.0.0.1:8123
```
